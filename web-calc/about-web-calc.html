You can calculate "%, vw, vh, em, rem", metallic means ratio and aspect ratio.「%, vw, vh, em, rem」の計算、貴金属比やアスペクト比の計算ができます。

Font size calculationフォントサイズの計算

  • "%, em" is relative to the parent element's font size.
    "rem" is relative to the root element (html), not the parent element.
    The value output to "em, rem" can also be used to calculate the line-height property.    「%、em」は親要素のフォントサイズからの相対値です。
    「rem」は親要素ではなくルート要素（html）に対して相対的です。
    「em、rem」に出力された値はline-heightプロパティの計算にも利用できます。
  • "vw (viewport width)" is relative to the screen width (including the scroll bar).
    "vh (viewport height)" is relative to the screen height (including the scroll bar).
    This is useful when you want to display the same number of characters on any device on a smartphone.
    Note: Set up media queries appropriately.    「vw（ビューポートの幅）」は画面の幅（スクロールバー含む）に対する相対値です。
    「vh（ビューポートの高さ）」は画面の高さ（スクロールバー含む）に対する相対値です。
    これは、スマートフォン上の任意のデバイスで同じ文字数を表示する場合に便利です。
    注意：メディアクエリの設定は適切に行ってください。

The metallic means and aspect ratio貴金属平均とアスペクト比

The calculation result is rounded down to four decimal places.
When converting to an integer, please follow the respective policy.計算結果は小数点4桁切り捨てです。
整数に変換する時には、それぞれの方針に従って処理してください。

The metallic means (metallic ratio)貴金属平均（貴金属比）

  • The golden ratio is the ratio that humans feel most beautiful.
    It's used for historical buildings, an art-object, business cards, and company logos.    黄金比は人間が最も美しいと感じる比率です。
    歴史的建造物や美術品、名刺、各社ロゴマークに使用されています。
  • The silver ratio is a ratio that has been beautiful in Japan since ancient times.
    Used for paper standards (A series and B series), Japanese anime characters, and ancient Japanese buildings.    白銀比は日本で古くから美しいと感じる比率です。
    用紙の規格（A判、B判）、日本のアニメキャラクター、日本古来の建造物に使用されています。
  • The bronze ratio is one of the metal ratios, but unfortunately it isn't used much.
    It might be good to use it for the main image of the homepage.    青銅比は貴金属比の一つですが、残念ながらあまり使われていません。
    トップページのメイン画像に使用するのもいいかもしれません。
