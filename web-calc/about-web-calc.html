Font size calculationフォントサイズの計算字体大小计算

  • "%, em" is relative value from the font size of the parent element.
    "rem" is relative to the root element (html), not the parent element.
    The values output in "em, rem" can be used to calculate the line-height property.    「%、em」は親要素のフォントサイズからの相対値です。
    「rem」は親要素ではなくルート要素（html）に対して相対的です。
    「em、rem」に出力された値はline-heightプロパティの計算にも利用できます。    "%, em "是来自父元素字体大小的相对值。
    "rem "是相对于根元素（html），而不是父元素。
    在 "em, rem "中输出的值可用于计算行高属性。
  • The "vw (viewport width)" is a relative value to the screen width (including scroll bars).
    The "vh (viewport height)" is relative to the screen height (including scroll bars).
    This is useful if you wanna display a same character length on any devices on smartphones.
    Note: Please set media queries appropriately.    「vw（ビューポートの幅）」は画面の幅（スクロールバー含む）に対する相対値です。
    「vh（ビューポートの高さ）」は画面の高さ（スクロールバー含む）に対する相対値です。
    これは、スマートフォン上の任意のデバイスで同じ文字数を表示する場合に便利です。
    注意：メディアクエリの設定は適切に行ってください。    "vw（视口宽度） "是相对于屏幕宽度的（包括滚动条）。
    "vh（视口高度）"是相对于屏幕高度的（包括滚动条）。
    如果你想在智能手机的任何设备上显示相同的字符长度，这很有用。
    注意：请适当地设置媒体查询。
